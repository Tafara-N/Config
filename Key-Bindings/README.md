## Location

> **Note:** Replace `<YourUsername>` | `$USER` with your user name.

**Windows:**

C:\Users\<YourUsername>\AppData\Roaming\Code\User\keybindings.json

**Ubuntu:**

`/home/$USER/.config/Code/User/keybindings.json`

## Optional
